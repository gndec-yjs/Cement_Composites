Appendix B: Illustrative Example of Analytical Model

Step-by-Step Calculation for Moment Coefficient

Given Parameters:

  • Reinforcement Index (ω): 0.314
  • Steel Fibre Volume Fraction (Vf): 0.75%
  • Concrete Compressive Strength (fck): 25 MPa
  • Beam Width (B): 100 mm
  • Beam Depth (D): 100 mm
  • Fibre Index (β): 0.045
  • Depth Ratio (m/D): 0.4
  • Depth Ratio (d'/D): 0.25
  • Neutral Axis Coefficient (ξ): 1 (for SFRC)

Analytical Model Equation

M_u/(f_ck * B * D^2) = 0.24 * ξ * (h_1/D)^2 + (h_2/D)^2 * (β/2) 
                    - (β/2) * (h_2/D) * (m/D) + (β/6) * (m/D)^2 
                    + f_l * γ * ω * (h_2/D) * (1 - (d'/D) * (D/h_2))

Step 1: Calculate h1/D and h2/D

Calculate h1/D:

h_1/D = 2.4 * [(β * (1 - m/2D) + f_l * γ * ω) / (ξ + 2.4 * β)]
Substituting the values: 
h_1/D = 2.4 * [(0.045 * (1 - 0.4/2) + 0.3025 * 0.87 * 0.314) / (1 + 2.4 * 0.045)]
Calculating:
  • 1 - 0.4/2 = 0.8
  • 0.045 * 0.8 = 0.036
  • 0.3025 * 0.87 * 0.314 ≈ 0.07974
  • 1 + 2.4 * 0.045 ≈ 1.108
Thus, 
h_1/D ≈ 2.4 * [(0.036 + 0.07974) / 1.108] ≈ 0.251

Calculate h2/D:

h_2/D = 1 - h_1/D ≈ 1 - 0.251 = 0.749

Step 2: Substitute Values into Moment Coefficient Equation

M_u/(f_ck * B * D^2) = 0.24 * 1 * (0.251)^2 + (0.749)^2 * (β/2) 
                     - (β/2) * (0.749) * (0.4) + (β/6) * (0.4)^2 
                     + f_l * γ * ω * (0.749) * (1 - (0.25/D) * (D/0.749))
Calculating each term:
  • 0.24 * (0.251)^2 ≈ 0.01512
  • (0.749)^2 * (β/2) ≈ 0.0126
  • - (β/2) * (0.749) * (0.4) ≈ -0.0009
  • (β/6) * (0.4)^2 ≈ 0.003
  • 0.3025 * 0.87 * 0.314 * (0.749) ≈ 0.0458
Putting it all together: 
M_u/(f_ck * B * D^2) ≈ 0.01512 + 0.0126 - 0.0009 + 0.003 + 0.0458 ≈ 0.07582

Step 3: Calculate Maximum Moment Mu

Use the maximum moment expression derived earlier: 
M_u = 0.07582 * (f_ck * B * D^2)
Calculating:
  • f_ck * B * D^2 = 25 * 100 * 10000 = 25000000 N.mm
  • M_u = 0.07582 * 25000000 ≈ 1895500 N.mm or 1895.5 kN.mm

Summary of Results

  • Moment Coefficient Mu/(fck * B * D2): 0.07582
  • Maximum Moment Mu: 1895.5 kN.mm
